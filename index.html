HTML

  • HTML stands for "Hypertext Markup Language"
  • HTML acts as the structure or framework of our web page
  • HTML is one of the three cornerstone languages used in web development

CSS

  • CSS stands for "Cascading Style Sheets"
  • CSS is used for adding visual style and aesthetic to your web page
  • CSS is the piece of your web page coding that is viewed directly by the web page users
  • CSS is one of the three cornerstone languages used in web development
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Using a ".card" in your style sheet will allow you to choose which elements of the HTML will be affected by information following the ".card"
  • To change the appearance or location of text on the web page, we can insert "font-size" or "text-align" underneath the corresponding HTML section that we want to edit

Git

  • git stands for "Global Information Tracker"
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used to make the web page interactive and easily navigated by the web page user
  • The use of JavaScript can be both on the client-side and the server-side
  • JavaScript is one of the three cornerstone languages used in web development
  • In JavaScript we can write and use functions that execute an action when we call it rather than it running automatically
  • Control flow is the order in which a computer executes a code in a script
  • A variable is a named container that allows us to store data in our code
  • "Math.floor()" and "Math.random()" can be used together to generate a random number
